Descrição do Projeto

O projeto tem como objetivo gerenciar produtos de forma organizada, podendo deletar, adicionar e controlar suas quantidades, cada item tem seu ID especifico assim como seu token para controle.

O projeto se utiliza de JavaScript e Typescript, Node.js, Angular, Express.js, Mongo, Html, CSS, além do Docker usado para apresentar este projeto.
